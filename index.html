When I wasn't studying, I contributed to various research projects by implementing simulation algorithms, usually taking advantage of the Compute Canada high-performance CPU clusters.

Vicsek Model Simulation

Particle swarming simulation made with JavaScript and WebGL. Particles tend to follow the average direction of their nearby neighbours but fluctuate randomly depending on the level of noise. More information about the algorithm can be found here. See on CodePen for an interactive demo.

Self-Learning Galaga Script

Lua script that plays the classic arcade game Galaga for 2000 frames. The script is given random instructions on which buttons to press at each frame. Before each iteration of gameplay, the script changes its instructions slightly and accepts the change if it leads to an increased score. The script can beat the first level in about 80 iterations.

Lattice Dipole Monte Carlo

Markov-chain Monte Carlo C++ code which simulates the thermodynamics for a system of magnetic dipoles with rotational freedom on a lattice. The code can be run on a high-performance computing cluster using MPI to simulate independent systems in parallel. Each simulation produces samples representative of uncorrelated measurements for a system in thermodynamic equilibrium. Data produced by this code were used in this publication.
